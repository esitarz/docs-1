Angularjs: Connecting To Your Organization

Published: Mon 19 March 2018
By OrderCloud.io

In Frameworks And Sdks.

tags: angularjs

Linking your front-end application to an OrderCloud seller or buyer organization can be accomplished using the scope and clientid constants described below.

Application Constants

src/app/app.constants.json is a small, yet powerful file host a JSON object of key value pairs to be used throughout the application. From this object, the build process will generate a file containing AngularJS constants: here, which can then be injected into your various AngularJS controllers, factories, providers, etc. Think of it as a base configuration or settings file for your application.
