Angularjs: Contributing

Angularjs: Contributing : frameworks-and-sdks
  • Mon 19 March 2018
  • frameworks-and-sdks
  • OrderCloud.io
  • #angularjs

The OrderCloud team welcomes any and all open-source contributors to create a pull request for bug fixes, enhancements, or new features (pending review).

Prior to writing any code, be sure to open an issue (Seller, Buyer) with a detailed description of your problem or proposed enhancement. We may already be on our way to delivering what you want!

We use GitHub's standard fork, branch, pull request workflow and expect any contributors to follow a similar workflow. Always provide a passing unit test for any fix or enhancement.

Thank you for being a part of the OrderCloud Community and helping make our resources the best they can be!
