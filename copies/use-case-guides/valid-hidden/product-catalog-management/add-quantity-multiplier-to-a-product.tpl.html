Introduction

This guide will teach you how to add a Quantity Multiplier to a product. The Quantity Multiplier is used to show the buyer user how many units per single line item they are ordering.

Please note the Quantity Multiplier is static numerical text that is displayed on the product detail page. It is not used to calculate total quantity of a line item.

Prerequisites

  • Create and Assign a Product

1. Creating a Quantity Multiplier for a Product

When creating a new product, simply set the QuantityMultiplier to the desired value. In our example, our exampleproduct is sold in quantities of 1000:

API reference: Create New Product
POST https://api.ordercloud.io/products HTTP/1.1
Authentication: Bearer put_access_token_here
Content-Type: application/json


{
  "ID": "exampleproduct",
  "Name": "Example Product",
  "QuantityMultiplier": 1000,
  "Active": true,
  "Type": "Static"
}

2. Update the Quantity Multiplier on an Existing Product

Now let’s pretend the quantity of units sold in our exampleproduct has changed from 1000 units per box to 1500 units per box. Since the product has already been created, we can easily update the QuantityMultiplier of our product using the Patch method. Start by entering the productID for the product being updated (exampleproduct) and only include the value you would like to Patch in the Request Body:

API reference: Update Product
PATCH https://api.ordercloud.io/products HTTP/1.1
Authentication: Bearer put_access_token_here
Content-Type: application/json


{
  "QuantityMultiplier": 1500
}

Conclusion

You should now be able to set a Quantity Multiplier on your products. Often times, your QuantityMultiplier will be set to 1 because the Quantity Multiplier is used to specify how many units are in each single line item.
