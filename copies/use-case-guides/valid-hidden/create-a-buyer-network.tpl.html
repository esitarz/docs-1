Introduction

This guide will teach you how to create a buyer network. A buyer network allows any number of buyers to order from a single application while still preserving their relationships to products and pricing as if they had their own application.

Learn more about buyer networks and when to use them from the Buyer Network blog post.

Prerequisites

  • Create an OrderCloud.io account

1. Create a Buyer Network

Begin by navigating to the dashboard and Select Buyer Networks from the left hand navigation:

2. Click the New Button.

Now simply provide a name for your buyer network and click ‘Create Buyer Network’.

Conclusion

Congratulations! You have created a buyer network. Now all of the buyer organizations you create within this seller application will be accessible using the same ClientID.

To try this out, go to your "Buyer Networks" tab again, click on your newly created buyer network and select Open API Console. Then, select the buyer organization you want to interact with from the list and start making API calls.
